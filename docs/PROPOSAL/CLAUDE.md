## Key Changes from Reality Check

The proposal has been updated based on empirical testing on Testnet Conway:

### Section 3: Technical Constraints & Reality Check

This section documents critical findings:

1. **GraphQL Limitations**
   - Schema doesn't load in Node Service
   - Cannot be used as primary API
   - Solution: REST API + CLI Wrapper

2. **Linera SDK Reality**
   - `linera-sdk` is only for Wasm compilation
   - No client SDK exists
   - Solution: CLI wrapper implementation

3. **Wallet Connector Availability**
   - MetaMask integration not verified for multisig
   - Dynamic not tested on Testnet Conway
   - Solution: Custom wallet implementation
